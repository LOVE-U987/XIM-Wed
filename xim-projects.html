01

[MOR] Major Overhaul and Reorganization

4/5 视觉完成度
7/10 整体评分
30+ 功能键
项目简介

这是一"坨"给偏技术玩家准备的高画质客户端整合包。官网已上线：modrinth.com/modpack/mor

1.1 项目背景

MOR（Major Overhaul and Reorganization）是一个专注于集成客户端优化、美学和可用性的整合项目。AI 测评给了 7/10，毒舌点在于："把它当高画质启动器而非即开即玩包，就不会踩坑"。

1.2 项目目标

打造一个视觉效果出色的高画质客户端整合包，为偏技术玩家提供个性化的游戏体验。

1.3 功能特点

高画质视觉效果

视觉完成度 4/5，截图即壁纸，为玩家提供沉浸式的游戏视觉体验

性能表现

不是极端 FPS 向，是特效向。1060+8G 以下别来，核显直接 30 FPS 献祭

丰富的功能键

30+ 功能键，提供丰富的操作选项，满足技术玩家的个性化需求

中文支持

提供中文汉化资源包（MOR-Setting），但必须版本完全对应

1.4 团队成员

陌路Molu

项目负责人

负责MOR主整合包的开发，专注于客户端优化与美化核心功能

Hzl

核心开发

参与MOR主整合包的开发工作

1.5 技术栈

Java Minecraft Fabric 1.20.1+ Gradle Git ShaderPack

1.6 成果展示

  • 已在Modrinth平台正式上线
  • 在BBSMC平台提供了镜像下载
  • MOR Setting 汉化资源包
  • MOR Error 问题反馈
  • 提供了完整的中文汉化方案
