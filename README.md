## 主要更新内容

### 1. 全新XIM文档页面设计

#### 1.1 三栏式现代化布局
- **左侧边栏**: 可折叠的文档目录导航，支持拖拽调整宽度(200px-400px)
- **主内容区**: 优化的阅读体验，最大宽度900px，清晰的内容层级
- **右侧大纲**: 当前页面标题快速导航（屏幕<1400px时自动隐藏）

#### 1.2 顶部工具栏
- 全局搜索框（支持快捷键 Ctrl+K）
- 字体大小控制（80%-150%可调）
- 主题切换（深色/浅色）
- 布局模式切换（标准/紧凑）
- 返回主页按钮

#### 1.3 交互功能
- 阅读进度条（顶部实时显示）
- 返回顶部按钮（滚动超过300px显示）
- 平滑滚动到锚点
- 键盘快捷键支持（ESC关闭侧边栏、/快速搜索）

#### 1.4 响应式设计
- Desktop XL (>1400px): 三栏布局
- Desktop (1024-1400px): 两栏布局
- Tablet/Mobile (<1024px): 单栏布局，侧边栏变为抽屉式

#### 2.1 实时搜索
- 输入时自动搜索，300ms防抖

## 使用说明

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl + K` | 聚焦搜索框 |
| `/` | 快速搜索 |
| `Esc` | 关闭侧边栏/搜索结果 |
| `↑` / `↓` | 在搜索结果中导航 |

### 个性化设置
- 字体大小: 点击工具栏的 A+ / A- 按钮
- 主题切换: 点击工具栏的月亮/太阳图标
- 布局模式: 点击工具栏的布局切换按钮
- 侧边栏宽度: 拖拽侧边栏右边缘调整
